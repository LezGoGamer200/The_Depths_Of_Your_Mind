Alright, first things first,
If you want to download & play the game, you will first need to make sure that you have everything installed that is required to play the game. Follow the folling steps to make sure everything is installed.

  1. Install the latest version of Python. Click here to go to their website.
  2. Make sure that that the following python modules are installed.
    Go here to find out how to install them.
    1. random
    2. os
    3. time
    These modules should be installed by default, but you should check just in case.
  3. Go to this Github repository & download the 'Dragons & Daydreaming - Better Inventory.py' file.
  4. Locate the file in your file directory, double click on it, and enjoy!
